# mdfplot
mdf文件曲线绘图工具，使用开源第三方库`mdflib`来解析mdf数据；使用`qcustomplot`来绘制曲线图。


## 第三方库
* [mdflib](https://github.com/ihedvall/mdflib) 
* [qcustomplot](https://www.qcustomplot.com/)
